# Moog Oracle Deployment

## Setup

```bash
git clone https://github.com/cardano-foundation/hal.git
cd hal/docs/deployment/moog/oracle
# copy the secrets
MOOG_VERSION=v0.3.2.0 docker compose up -d
```
